🔧 Enable Cross-Device Sync

To sync bookings across all devices in real-time, you need a free Firebase project. It takes about 3 minutes to set up.

  1. Go to console.firebase.google.com
  2. Click Add project → name it casa-labrador-portal
  3. Go to Build → Firestore Database → Create (Test mode)
  4. Go to Project Settings → Your Apps → Web → Register App
  5. Copy the firebaseConfig object and paste it into this HTML file, replacing the placeholder at the top of the <script type="module"> block
  6. Save the file and open it on any device — bookings will sync instantly! ✅
